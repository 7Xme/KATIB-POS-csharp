# Ketaba POS - User Guide

## Getting Started

### Login
1. Launch the application
2. Enter username: `admin`
3. Enter password: `admin123`
4. Click "Login"

## Main Modules

### Dashboard
- View KPI cards: Today's Sales, Transactions, Low Stock Items, Active Loans
- Recent activity feed shows latest transactions

### POS (Point of Sale)
1. Search products by name or barcode in the search bar
2. Click "Add" on products to add to cart
3. Adjust quantity directly in cart
4. Enter amount paid
5. Click "Complete Sale" to finalize

### Products Management
- View all products in a data grid
- Search and filter by category
- Pagination for large inventories
- (Future: Add/Edit/Delete product forms)

### Customers & Suppliers
- View and search customer/supplier lists
- Track balances and credit limits

### Sales History
- View all completed sales
- Filter by date range
- Print receipts

### Loans Management
- Track customer and supplier loans
- View amounts, paid amounts, and status
- Mark loans as paid

### Settings
- Configure company name (English & Arabic)
- Set tax rate
- Change currency
- Backup and restore database

## Keyboard Shortcuts
- F1: Help
- F2: New (future)
- F3: Search
- F4: Save
- ESC: Cancel

## Theme Switching
- Click the toggle switch in the top-right corner to switch between light and dark themes
- Settings persist across sessions

## Backup & Restore
1. Go to Settings
2. Click "Backup Database" - choose save location
3. Click "Restore Database" - select backup file
4. Restart the application after restore

## Language Support
- Full Arabic (RTL) support
- Arabic names for products, categories, customers, suppliers
- Language switching via Settings
